# 🎬 ParkyFilms - Sistema de Gestión Cinematográfica

Aplicación web full-stack desarrollada para la gestión integral de un sistema de cine, incluyendo autenticación de usuarios, reservas de boletos, administración de películas y análisis básico de datos.

## 🚀 Descripción del Proyecto

Este proyecto fue desarrollado como una solución completa para la administración de un sistema cinematográfico, permitiendo gestionar usuarios, funciones de cine y reservas en tiempo real.

Se implementaron funcionalidades clave como autenticación segura mediante JWT y OAuth con Google, así como integración con base de datos PostgreSQL para el manejo eficiente de la información.

## 🧠 Tecnologías Utilizadas

- Frontend: Angular, TypeScript
- Backend: Node.js, Express.js
- Base de datos: PostgreSQL
- Autenticación: JWT, OAuth 2.0
- Despliegue: Railway

## 📊 Resultados

- Implementación de un sistema completo de reservas funcional
- Integración de autenticación segura con Google OAuth
- Diseño de base de datos relacional para gestión de usuarios, películas y reservas
- Desarrollo de funcionalidades escalables para administración y análisis básico
